Aprofundei o uso de estruturas de repetição em Python:

1. Laço FOR:
   - Usei o loop for para repetir blocos de código um número determinado de vezes.

   - Aprendi que a função "range(n)" gera uma sequência de 0 até n-1.

   - Usei o "for" com "print()" para contar e exibir valores na tela.

2. Laço WHILE:
   - Usei o "while" para repetir enquanto uma condição for verdadeira.

   - Controlei o loop com uma variável "contador" e usei incremento com += 1.

3. Comando BREAK:
   - Interrompe imediatamente um loop, mesmo que ainda restem repetições.

   - Usei dentro de um for para sair quando uma condição específica foi atingida (ex: i == 5).

4. Comando CONTINUE:
   - Pula o restante do código dentro do loop e vai para a próxima iteração.

   - Usei para ignorar uma repetição específica sem sair do loop.

5. Tarefas Práticas:
   - Criei programas que contam progressivamente com for e regressivamente com while.

   - Combinei as duas estruturas no mesmo código para praticar a lógica de repetição.
