1. Função que recebe uma lista como parâmetro:
Permite trabalhar com vários dados de uma vez dentro da função, usando "for" para percorrer os itens.
Ex.:

def exibir_produtos(lista):
    for produto in lista:
        print(f"{produto['nome']} custa R${produto['preco']:.2f}")

2. Função que retorna um dicionário:
Permite criar dicionários dentro da função e devolvê-los com "return":
Ex.:

def criar_contato(nome, telefone):
    contato = {
        'nome': nome,
        'telefone': telefone
    }
    return contato

3. Função com vários parâmetros
Trabalhei com funções que contém mais de dois parâmetros:
Ex.:

def cadastrar_livro(titulo, autor, ano):
    return {
        'título': titulo,
        'autor': autor,
        'ano': ano
    }

4. Função dentro de um laço:
Usei funções dentro de `for`, principalmente para cálculos ou conversões:
Ex.:

for pessoa in pessoas:
    imc = calcular_imc(pessoa['peso'], pessoa['altura'])
    print(imc)
